# alexcrist.github.io

My personal website, [check it out](http://alexcrist.github.io)! All homegrown HTML, CSS, and JS with a little help from JQuery.

## Structure

    .
    ├── assets/               # Folder for any static assets such as images
    ├── css/                  # Folder for CSS files
    ├── data/                 # Folder for static data files
    |   └── content.json      # Contains the content of the sections of the site
    ├── js/                   # Folder for JS files
    |   └── content.js        # Populates the page with the data from content.json
    ├── vendor/               # Folder for any external libraries
    └── index.html            # An HTML scaffold of the page that is populated with content from content.js
